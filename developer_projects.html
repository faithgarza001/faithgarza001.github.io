InsightBridge: Enterprise BI Dashboard

Overview: InsightBridge is a full-scale, enterprise-grade analytics dashboard designed for decision-makers who need clean, fast, and actionable insights. Developed in ASP.NET Core with a SQL Server backend, the platform integrates Power BI Embedded for interactive visualizations and enables data-driven decision-making across departments. Key Features: Multi-source data aggregation with SQL optimization Secure, role-based access and module permissions Interactive, exportable dashboards and KPIs Real-time filtering and responsive UI.

ForecastIQ: AI-Powered Sales Predictor

Overview: ForecastIQ is an intelligent forecasting tool that leverages ML.NET and time-series regression models to predict future sales trends based on historical data. Built using ASP.NET Web API and integrated with Azure Functions for scheduled inference runs, this tool empowers sales teams with foresight. Key Features: Historical sales data ingestion and cleanup Machine learning regression pipeline Forecast visualization and comparison tool JSON-based REST API endpoints for predictions

PulseTrack: Real-Time Healthcare Monitoring

Overview: PulseTrack is a real-time vitals and appointment monitoring system tailored for small clinics and healthcare institutions. Built with Blazor Server and SignalR, it allows healthcare professionals to monitor patient activity, communicate securely, and respond to abnormal patterns quickly. Key Features: Vitals dashboard with dynamic thresholds and alerts SignalR-based real-time updates and staff communication Role-specific dashboards for nurses, doctors, and admins PDF summaries and HIPAA-aware data handling

NeuroSort: AI + Servo-Powered Sorting Arm

Overview: AutoSortBot is a mechatronic system that classifies and sorts physical items using a camera sensor and servo mechanisms. The control software is written in C# using .NET with serial communication to Arduino/Microcontroller firmware. Key Features: Hardware integration with Arduino/Microcontroller via Serial Port Live camera feed and AI-based object detection Servo-driven sorting arm based on classification UI built with WinForms/WPF to monitor and control operations

SmartInvoice.AI: Automated Invoice Processor

Overview:SmartInvoice.AI streamlines finance operations by allowing users to upload scanned or digital invoices. Using Tesseract OCR and OpenAI/Azure Cognitive Services, it extracts invoice data and automatically classifies vendors, dates, and totals. The system integrates with accounting software via API. Key Features: Drag-and-drop UI for PDF upload OCR and AI-powered field extraction Vendor classification using NLP Export to accounting tools like QuickBooks

ServiceMesh360: Microservices & Scalable Logistics

ServiceMesh360 is a distributed logistics management system built using a microservices architecture with .NET Core, Docker, and RabbitMQ. Designed to mimic enterprise supply chain operations, each service (Orders, Inventory, Notifications, Billing) is independently deployable, enabling CI/CD and horizontal scaling. Key Features: Event-driven messaging with RabbitMQ and MassTransit Independent services: Auth, Inventory, Billing, and Notifications API Gateway with Ocelot for centralized routing Containerized deployment via Docker Compose / Kubernetes

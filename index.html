Featured Projects

MechaNest Homes

"Tap, trigger, transform—smart home systems made tangible."
Control your smart environment with a tap. This 3D-printed home model uses NFC tags and Arduino logic to simulate real-time smart automation—lighting, fan, and door systems—all powered through a custom dashboard UI built with accessibility in mind.

CoinQuest: UE5 Blueprint Adventure

"Where Code Meets Creativity in a Thrilling Coin Hunt!"
A fast-paced, third-person coin collection game built with Unreal Engine 5 using Blueprints. Through hands-on development, I explored variable logic, conditionals, UI interface design, asset integration, and basic AI behavior. This project strengthened my understanding of how programming principles apply to game mechanics and player experience—bridging code and creativity in a real-time 3D engine environment.

DataForge BI

"Transforming Queries into Clarity."
A scalable, containerized business intelligence solution powered by Apache Superset. Deployed using Docker for rapid setup and portability, this project features customized dashboards, role-based access, and advanced SQL-based visualizations. Designed for teams needing actionable insights from complex datasets.

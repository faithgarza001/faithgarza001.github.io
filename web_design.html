Featured Projects
MechaNest Homes
"Tap, trigger, transform—smart home systems made tangible."
This project showcases a futuristic 3D-printed smart home prototype, integrating advanced mechatronics with intuitive control systems. It features NFC-triggered actions that seamlessly automate various home functions, such as lighting, climate control, and door access. The system is powered by Arduino microcontrollers, connected to a custom-built, accessible dashboard interface. This allows users to effortlessly control and customize their living environment through touch-driven interactions, demonstrating practical applications of IoT, embedded systems, and responsive UI/UX design in a tangible, interactive model.
CoinQuest: UE5 Adventure
"Where Code Meets Creativity in a Thrilling Coin Hunt!"
This project involved developing an interactive 3D adventure game within Unreal Engine 5, focusing on core programming concepts implemented through visual scripting (Blueprints). I designed and integrated game mechanics such as player movement, collectible item interactions, and environmental puzzles. Key aspects included creating dynamic UI elements (HUDs), implementing player feedback systems, and scripting AI behaviors. This experience deepened my understanding of how fundamental programming logic drives interactive storytelling and real-time 3D experiences, from asset integration and level design to performance optimization within a game engine.
ERP NFC Smart Tag Login & Mobile Integrations
"Empowered Operations: NFC & Mobile for ERP."
A robust integration of NFC (Near Field Communication) smart tag authentication within an ERP ecosystem, enabling rapid, secure, and touch-based user login. Concurrently, it developed comprehensive mobile controls for critical ERP functionalities, utilizing RESTful API endpoints for real-time data synchronization. Features include mobile-optimized interfaces for inventory management, asset tracking, and field service operations, ensuring cross-platform compatibility and enhanced operational efficiency through streamlined data capture and immediate system updates. Focus was placed on secure communication protocols and intuitive UX design.
